# Task Briefs F5-F8

## F5-Backend-1

Feature:
`F5 ATS Scraping Pipeline`

Role:
`Backend`

What this task is:
Implement the scraper orchestration boundary and ATS detection integration.

Why it exists:
All provider-specific scraping work depends on one stable orchestration path.

Depends on:
- `F4-Backend-1`

Done when:
- one orchestration path exists for ATS detection and scraper dispatch

Out of scope:
- full provider coverage alone
