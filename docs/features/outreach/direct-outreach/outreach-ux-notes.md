# Direct Outreach UX Notes

## Minimal UX
- User clicks "Find contacts" on a job or company.
- User sees 2-3 contacts with validated emails and confidence.
- User clicks "Generate draft" and reviews the email.
- User sends from their own email client.

## Guardrails
- Explicitly state that CeeVee does not send emails.
- Require confirmation before logging outreach as sent.
