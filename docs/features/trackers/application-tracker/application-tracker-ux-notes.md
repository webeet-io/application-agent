# Application Tracker UX Notes

## Minimal UX
- A simple action: "Mark as Applied" on a job listing.
- A follow-up action: "Update outcome" with a dropdown.

## Outcomes
- interviewing
- rejected
- offer

`no_response` remains a planned follow-up outcome and is not accepted in this MVP API yet.

## Timing
- User updates are manual (no background automation in MVP).
- Optional later improvement: prompt after N days with no response.
